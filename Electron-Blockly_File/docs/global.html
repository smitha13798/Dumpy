insertCommentAtLine(filePath, viewName, lineIndex)

Inserts a comment at a specified line in a file. The comment will be in the format `#viewName+\n#viewName-` where `viewName` is provided as a parameter.
Parameters:
Name Type Description
filePath string The path to the file where the comment will be inserted.
viewName string The name to be used in the comment (used to generate `#viewName+` and `#viewName-`).
lineIndex number The index at which the comment should be inserted. The index should be 0-based.
Source:

Throws:
  • If there's an issue reading or writing to the file.
    Type
    Error
  • If the lineIndex is out of bounds of the file's total lines.
    Type
    RangeError
Example

insertCommentAtLine('/path/to/file.txt', 'MyView', 5);
